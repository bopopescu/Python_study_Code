# 1. 模块
- 一个模块就是一个包含Python代码的文件，后缀名就是.py就可以，模块就是Python文件
- 为神魔我们用模块
    - 程序太大，编写维护非常不方便，需要拆分
    - 模块可以增加代码重复利用的方式
    - 当做命名空间使用，避免命名冲突
- 如何定义模块
    - 模块就是一个普通文件，所以任何代码可以直接书写
    - 不过根据模块的规范，最好最好在模块中包含以下内容：
        - 函数（单一功能）
        - 类（相似功能的组合，或者类似业务模块）
        - 测试代码
        
- 如何使用模块
    - 模块直接导入
        - 加入模块名称直接以数字开头
    - 语法
            import  module_name
            module_name.function_name
            module_name.class
